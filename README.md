# CityRunner
City Runner is a free to play interactive game made in Unity, built for WebGL and deployed using GitHub pages.

## Controls -
```
To start -> Left Click on screen to start game.
Jump -> Space Bar
Double Jump -> Jump Bar + Jump Bar
Slide -> Shift
```
